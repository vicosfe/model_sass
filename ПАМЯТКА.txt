					Gulp
npm install- скачать все модули после скачивания проекта с репозитория. 
npm i name_module --save-dev При возникновании ошибки что отсутствует какой то модуль
gulp - запустить gulp (browser-sync)
gulp build-Front - сборка фронтенда. Запускается после каждой сделанной страницы
gulp build-Back - сборка всего проекта. Делается после окончания работ.
npm rebuild node-sass -  версия сас не совпадает с версией npm
